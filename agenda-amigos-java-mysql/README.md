# Agenda de Amigos

Proyecto académico de Acceso a Datos desarrollado en Java con interfaz gráfica y conexión a base de datos.

## Funcionalidades
- Alta de amigos
- Listado de registros
- Búsqueda por nombre o teléfono
- Modificación de datos
- Gestión básica de pandillas

## Tecnologías utilizadas
- Java
- Swing
- JDBC
- MySQL
- Eclipse

## Notas
Proyecto realizado con fines de aprendizaje dentro del módulo de Acceso a Datos.
