## 🎮 Descripción

**Agenda de Amigos** es una aplicación de escritorio orientada a la gestión de una agenda personal, permitiendo registrar amigos, almacenar sus datos y asociarlos a una pandilla.

El proyecto aplica una estructura basada en el patrón **DAO**, separando la lógica de acceso a datos de la interfaz gráfica para mejorar el mantenimiento y la organización del código.

---

## ✨ Funcionalidades

- Alta de amigos
- Listado de registros
- Búsqueda por nombre o teléfono
- Modificación de datos
- Asociación de amigos con una pandilla
- Persistencia de datos en MySQL

---

---

## 🧠 Diseño de base de datos

El modelo parte de dos entidades principales:

- **Amigo**
- **Pandilla**

La relación entre ambas es de **1:N**, donde una pandilla puede tener varios amigos y cada amigo pertenece a una única pandilla.
